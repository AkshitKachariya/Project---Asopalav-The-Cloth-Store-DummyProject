# 🛍️ ASOPALAV.IN - The Cloth Store

A modern and responsive E-commerce Website for men’s and women’s fashion.

Users can browse products, view product details, add items to cart, and place orders securely with a smooth shopping experience across all devices.

---

## ✨ Features

### 👤 User Panel
* User Registration & Login
* Browse Products by Categories
* Product Details Page
* Add to Cart Functionality
* Secure Order Placement
* Manage Profile & Account
* Contact / Message Admin

### 🛠️ Admin Panel
* Secure Admin Login
* Manage Categories
* Add / Update / Delete Products
* Manage Orders
* Manage Users & Messages
* Update Store Locations
* Manage Branding & Social Links

---
